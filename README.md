### 2. Write a program to find GCD (greatest common divisor or HCF) and LCM (least common multiple) of two numbers. 

> Test Data:

    Enter two numbers: 22 55
    
> Expected Output:

    GCD = 11
    LCM = 110

> Source Code:

```c
#include <stdio.h>

int main()
{
    int num1, num2, sm, gcd, lcm;

    printf("\nEnter two numbers: ");
    scanf("%d %d", &num1, &num2);

    sm = num1;

    if (num1 > num2)
        sm = num2;

    while (sm != 0)
    {
        if (num1 % sm == 0 && num2 % sm == 0)
        {
            gcd = sm;
            break;
        }
        sm--;
    }

    lcm = num1 * num2 / gcd;

    printf("\nGCD = %d", gcd);
    printf("\nLCM = %d", lcm);

    return 0;
}
```
